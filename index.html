Endpoints

GET /api/wschat/users/message/stream?since=n

n is the start id

Waits up to 3.5 minutes for a message. All messages available since the specified id, up to a server defined limit, are send as the response. You are expected to immediately request the next message, using the highest id you know, as the value for since.
